A small LUA script for Azerothcore that will give you a random amount of a random primary stat every time you level up.
By default the amount is between 1 and 4 and all the primary stats are enabled.
You can change these values in the script itself and you can disable undesired primary stats.
Run the two SQL queries after placing the LUA file in the lua_scripts folder.
Feel free to share/modify whichever way you see fit.
